# InterviewShare - Detailed Project Documentation
## Overview
InterviewShare is a full-stack platform for sharing interview experiences. It allows students to post their placement journeys, including company names, roles, and detailed descriptions of the interview process.

## Features
- **User Authentication**: Secure login/register using Flask-Security.
- **CRUD Operations**: Create, Read, Update, and Delete interview experiences.
- **Search**: Filter experiences by company or role.
- **Social Interaction**: Like/Unlike experiences.
- **Responsive UI**: Built with Vue.js 3 and a custom CSS design system.
